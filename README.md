# 🛒 E-Commerce Website

A simple and responsive **E-Commerce Website** developed as a **Summer Mini Project** using **PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap**. This project demonstrates the basic workflow of an online shopping platform, including user authentication, product browsing, shopping cart management, and checkout.

---

## 📌 Features

- 👤 User Login Authentication
- 🛍️ Browse Products
- ➕ Add Products to Cart
- 🔢 Update Product Quantity
- ❌ Remove Products from Cart
- 💰 Automatic Total Price Calculation
- 🛒 Checkout Functionality
- ✅ Order Confirmation Page
- 📱 Responsive User Interface

## 🚀 How to Run the Project

### Prerequisites

- XAMPP
- PHP
- MySQL
- Web Browser

### Installation

1. Clone the repository

```bash
git clone https://github.com/sazida-shaik/Ecommerce-Website.git
```

2. Move the project folder to the **htdocs** directory inside XAMPP.

3. Start **Apache** and **MySQL** using XAMPP Control Panel.

4. Open **phpMyAdmin**.

5. Create a database.

6. Import the SQL file from the `database` folder.

7. Open your browser and visit:

```
http://localhost/ecommerce
```
