# E-Commerce Website

A simple e-commerce web application developed as a summer mini project using PHP and MySQL. The project demonstrates the core functionality of an online shopping platform, including user authentication, product browsing, shopping cart management, and checkout.

## Features

- User Login
- Product Listing
- Add to Cart
- Update Product Quantity
- Remove Products from Cart
- Dynamic Total Price Calculation
- Checkout Page
- Order Confirmation

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap
